Este é um título

Como a imagem tem um z-index de -1, ela será colocada atrás do texto.

Nota: o z-index só funciona em elementos posicionados (position: absolute, position: relative, position: fixed ou position: sticky) e itens flex (elementos que são filhos diretos de elementos display: flex).

Outro exemplo utilizando Z-index

Um elemento com maior ordem de pilha está sempre acima de um elemento com menor ordem de pilha.

Box preto (z-index: 1)
Box cinza (z-index: 3)
Box verde (z-index: 2)
